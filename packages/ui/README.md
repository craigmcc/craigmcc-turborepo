# packages/ui

## Introduction

This package contains installed Shadcn-UI components, which are installed based
primarily on the instructions for a
[Shadcn-UI in a monorepo](https://ui.shadcn.com/docs/monorepo)installation.
It contains all of the available components, so you do not
need to worry about installing the individual ones you want in your own
application repo.
